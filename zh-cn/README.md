

PlatONE是基于隐私计算的新⼀代联盟区块链平台，可以⽀持企业级应⽤。平台提出了⼀种以隐私计算为特⾊的企业级联盟链基础设施，可满⾜⾦融商业等多种需求场景。

* [SDK](zh-cn/SDK/[Chinese-Simplified]-SDK介绍) ：该部分介绍了如何通过SDK与区块链进行交互。
